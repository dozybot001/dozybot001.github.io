本章你要掌握的主线

这一章其实就围绕一条“输入 → 系统 → 输出”的链路：

  • 图像（2D 信号）如何分解：把复杂图像写成很多“简单基”的叠加（例如冲激、正弦/复指数）。
  • LSI（线性 + 移位不变）系统：系统完全由它对“单位冲激”的响应决定。
  • 2D 卷积：LSI 系统的空间域计算工具。
  • 2D 傅里叶变换：把卷积变成乘法、把空间域问题转到频域看“滤波”。
  • 采样：连续图像如何变成离散像素；采样会导致混叠（aliasing），要懂 Nyquist 与抗混叠滤波。

频域视角怎么解释“滤波”？

  • 大的区域 → 那些频率保留
  • 低通：保留低频（平滑、去噪、模糊）
  • 高通：保留高频（边缘/细节增强、锐化）
  • 带阻/带通：去周期噪声、去条纹等

2D 频率直觉：

  • 频谱中心（常用 fftshift 后）附近 ≈ 低频（大尺度变化、亮度缓慢变化）
  • 远离中心 ≈ 高频（边缘、纹理、噪声）

采样与混叠（aliasing）：必须会画频谱“重复”图

连续信号采样在频域会产生周期重复（2D 同理：频谱在两个维度上复制平铺）。
如果复制后的频谱发生重叠，就会 混叠：高频“伪装”成低频，细节变成摩尔纹/锯齿等。

Nyquist 条件（2D 思路）：采样频率至少覆盖最高频率的两倍
（用角频率写就是 ；2D 在两个维度分别满足）

抗混叠（Anti-aliasing）：采样前先低通

采样前用低通滤波器限制带宽（模糊一点点但避免错误信息）：

  • 连续域：光学低通（相机 OLPF）
  • 数字域：下采样前做低通（例如 Gaussian blur 再缩小）

下采样 / 上采样（实现题常考）

下采样（decimation）：每隔 取一个点

  • 频域会“拉伸”并导致混叠风险 ↑
  • 所以必须：先低通再下采样

上采样（interpolation / upsampling）：插零再滤波（理想情况）

常见插值方法：nearest / bilinear / bicubic（越平滑越“低通”）。
