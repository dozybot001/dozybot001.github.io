Android Studio

Android 开发需要安装 Android Studio，https://developer.android.com/studio?hl=zh-cn

一并下载 Android Studio 安装包和下面的命令行工具。

Android SDK

安装 Android Studio 后它会提示你下载并安装 Android SDK，然而下载过程可能十分艰难，动不动就握手失败，安装完成后他会提示你有哪些安装失败了，点击 Show Details，看哪些文件下载失败，他会给出 https://dl.google.com/android/repository … 开头的链接 . 直接手动打开浏览器下载。之后将下载的文件放在C:\Users\{UserName}\AppData\Local\Android\Sdk文件夹中。

注意：

  1. 通过 Android Studio 下载 SDK 失败的文件，也会创建文件夹，里面会存在.install\.installData文件。
  2. 通过浏览器手动下载好压缩包后，将解压缩出来的文件夹打开，进入有大量不同类型文件的目录，复制所有文件，将其放在与 .install 文件夹同级的目录。
  3. 之后再运行 Android Studio 检查，它还会提示缺失 SDK，如果文件位置正确，再走一遍下载流程，Show Details 会提示没有文件需要更新，之后再打开 Android Studio 就不会提示缺失 SDK 了。

完整 SDK 文件夹结构：

将下载的命令行工具解压缩也放在了这个目录：参考 https://developer.android.com/tools/sdkmanager?hl=zh-cn

  1. 从 Android Studio 页面下载最新的命令行工具软件包，然后提取该软件包。
  2. 将解压缩的 cmdline-tools 目录移至您选择的新目录，例如 android_sdk。这个新目录就是您的 Android SDK 目录。
  3. 在解压缩的 cmdline-tools 目录中，创建一个名为 latest 的子目录。
  4. 将原始 cmdline-tools 目录内容（包括 lib 目录、bin 目录、NOTICE.txt 文件和 source.properties 文件）移动到新创建的 latest 目录中。现在，您就可以从这个位置使用命令行工具了。
