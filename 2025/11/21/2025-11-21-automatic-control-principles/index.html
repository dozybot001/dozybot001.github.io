前言

个人自制学习资料，无商业用途。

绘制原理框图：

  1. 找出被控对象和被控量，一般在题干给出
  2. 找出输入信号和输入变换装置，一般与被控量类型一致
  3. 找出连接输入、输出的反馈装置
  4. 反馈装置输出的信号与输入变换装置的信号比较，作为比较装置
  5. 比较装置后依次连接其他装置

信号流图-Mason 公式求传函：

  1. “ 输入、输出、干扰、比较点、分支点 “ 标点，画出信号流图
  2. 检查每个反馈，负反馈需乘-1
  3. 找出所有回路，写出增益
  4. 求出，表示任意 2 个 “ 互不接触回路 “ 增益之乘积
  5. 找出所有前向通路，写出增益，与去除该通路的值
  6. 根据 Mason 公式，求出传函

闭环传递函数-Routh 判据：

  1. 写出闭环传递函数，也就是总传递函数

  2. 取出分母，令 =0，得到特征方程

  3. 先看特征方程系数是否有 0 或者负数，一旦有则必不稳定

  4. 如果全为正数，则写 Routh 表

  5. 先列写，共行

  6. 然后用系数上到下，左到右填充前两行

  7. 通过规则计算其他行数据：

    下一行第 i 列数据，由上两行第 1 列 2 数和第 i+1 列 2 数，共 4 数算得

    ，分子为上两行 4 个数组成的 2 阶行列式，分母为行列式左下角数

    非首列缺项用 0 代替，首列计算得 0 用极小正数代替

    某行全为 0，用上一行数据组成辅助方程求导后的系数代替

    所有规则均为排 Routh 表服务，不能继承到后面的判定

  8. 观察 Routh 表：

    当首列元素非 0 且无符号改变，则系统稳定

    出现 0，说明存在关于原点对称的一组根

    如果该行其他元素不为 0，则为纯虚根，临界稳定；

    如果该行其他元素均为 0，则不为纯虚根，不稳定

    有符号改变，改变几次，则右半平面有几个根，不稳定

开环传递函数-稳态误差：

  1. 写出开环传递函数，注意是开环不是闭环
  2. 系统类型 = 开环传递函数分母单的阶次
  3. 将输入信号拆分成多个基础信号，阶跃、速度、加速度
  4. 根据基础信号类型计算相应误差系数，非 0 有限误差系数就是增益
  5. 根据公式求出稳态误差之和，P77

闭环传递函数-根轨迹：

  1. 写出闭环传递函数，也就是总传递函数
  2. 取出分母，令 =0，得到特征方程
  3. 将特征方程化成的形式，可以用画出关于参数的根轨迹，一般为增益因子
  4. 化为零、极点形式，在图上 x 标极点，o 标零点
  5. 起：轨迹起始于极点，终止于零点，只有个极点终止于个零点，个极点终止于无穷远
  6. 分：轨迹共有个分支
  7. 对：轨迹关于实轴对称
  8. 渐：有条渐进线指向无穷远，渐进中心，渐进倾角
  9. 实：确定实轴上的根轨迹，根轨迹右边的零、极点数之和为奇数
  10. 分：确定分离点、会合点，写出参数的表达式，对的导数为 0，求出根，在根轨迹上的根为分离点、会合点
  11. 出：确定出射角，某极点出射角 + 其他极点到该点的角 =+ 零点到该点的角
  12. 虚：确定虚轴交点，利用特征方程列劳斯表，用使得某一行首列为 0 的，该行上一行作辅助方程，令 =0，求出一组纯虚根
