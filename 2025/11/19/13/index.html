测量及测量系统

概述

  1. 测量的定义：借助测量仪器，通过某种方法将被测量与参考基准比较，从而得到量化结果的实验过程。
    三要素：测量仪器、测量方法、测量对象

  2. 测量结果=估计值$\pm$不确定度 单位 （P=95%）。
    估计值常用平均值，不确定度为K倍标准差，K与置信概率P有关

  3. 国际单位制（SI）的7个基本单位：$m\ s\ kg\ A\ K\ mol\ cd$

  4. 测量仪器基本功能：给出测量示值，提供参考基准（其准确程度决定结果准确度上限）

  5. 测量仪器基本要求

    1. 准确：误差小
    2. 精密：不确定度小
    3. 可靠：可等精度重复
    4. 灵敏：能反应小变化

  6. 测量方法选取原则：科学、合理、方便

  7. 测量方法分类

    1. 按测量结果的获得方式：直接测量、间接测量（测量结果经计算得到）
    2. 按被测量所处状态：稳态测量、暂态测量
    3. 按设备在线与否：在线测量、离线测量（需脱离工作才能测量）

  8. 现代数字化测量系统的基本组成：传感器->信号调理->模数转换->计算机->显示

2、静态特性

  1. 静态特性的定义：输入x稳态时，输出y与x的关系。
    静态特性又称”刻度特性””标准曲线””校准曲线”

  2. 静态特性的获得：规定工况下获得

  3. 静态特性表现形式：数表 or 标定曲线 or 数学表达式（固有特性 or 拟合特性）

  4. 基本特征参数

    1. 灵敏度：$$S = \frac{dy}{dx}$$
    2. 线性度：定量反映标定曲线偏离拟合直线的程度（线性度只对线性系统而言，实测->标定曲线）

$$linearity = \frac{B_{\max}}{A} \times 100\%$$
  1. 零位：零输入时测量值
  2. 分辨力：测量仪表能分辨的最小输入变化（一个”最低有效位”LSB）

  1. 分辨率：分辨力与满量程之比
  2. 回程误差 or 滞回误差：量程内，输入增减变化标定曲线静态特征的不一致程度

$$Backhaul\ error = \frac{h_{\max}}{A} \times 100\%$$
  1. 量程：仪器标称范围上下限之差
  2. 动态范围：最强输入与最弱输入分贝差
$$Dynamic\ Range = 20\lg\frac{I_{\max}}{I_{\min}}$$

动态特性

  1. 一阶系统

    1. 微分方程：$\tau\dot{y} + y = kx$
    2. 传递函数：$H(s) = \frac{Y(s)}{X(s)} = \frac{k}{\tau s + 1}$
    3. 频率特性（$s = j\omega$）：$H(j\omega) = \frac{k}{j\omega\tau\ + \ 1} = k\left| H(j\omega) \right|e^{- j\varphi}$
    4. 幅频特性：$\left| H(j\omega) \right| = \frac{1}{\sqrt{1 + (\omega\tau)^{2}}}$
    5. 相频特性：$\varphi(\omega) = - \arctan(\omega\tau)$
    6. 直流放大倍数$k$，时间常数$\tau$，转折频率$\omega_{0} = \frac{1}{\tau}$

  2. 二阶系统

    1. 微分方程：$\frac{1}{\omega_{0}^{2}}\ddot{y} + \frac{2\xi}{\omega_{0}^{2}}\dot{y} + y = kx$
    2. 传递函数：$H(s) = \frac{Y(s)}{X(s)} = \frac{k}{\frac{1}{\omega_{0}^{2}}s^{2} + \frac{2\xi}{\omega_{0}^{2}}s + 1}$
    3. 特征方程：$s^{2} + 2\xi s + \omega_{0}^{2} = 0$
    4. 阻尼角：$\theta = arccos\zeta$
    5. 时间常数：$T' = \frac{1}{\zeta\omega}$
    6. 阻尼振荡角频率：$\omega_{d} = \omega_{0}\sqrt{\left| \zeta^{2} - 1 \right|}$
    7. 上升时间：$t_{r} = \frac{\pi - \theta}{\omega_{d}}$
    8. 峰值时间：$t_{p} = \frac{\pi}{\omega_{d}}$
    9. 超调量：$M_{p} = e^{- \frac{t_{p}}{T'}}$
    10. 调整时间：$t_{s} = 3T'\ or\ 4T'$

误差与测量不确定度

基本概念

  1. 标称值：测量器具（被测物）上标注的量值（一般附加准确度等级和误差范围）
  2. 示值：测量仪器给出的值
  3. 准确度：测量结果与真值一致的程度（定性概念）
  4. 定量描述：准确度等级&不确定度
  5. 等精度重复测量：相同条件下对同一被测量的多次连续测量
  6. 真值：客观存在但不可测的真实值
  7. 理论真值：可以计算或推导出来
  8. 相对真值：满足准确度的近似真值（例：标准仪器的参考基准，更高精度仪器的测量值）
  9. 约定真值：人类最高水平所复现的单位基准，以法律形式规定（基本单位基准值）

表达误差的形式

  1. 按性质分类：系统误差（已定、未定），随机误差，粗大误差
  2. 描述测量行为的误差：绝对误差，相对误差
  3. 仪表的误差：最大允许误差，最大引用误差（线性度，回程误差，迟滞误差，基本误差）

后四种误差介绍

涉及计算的真值用约定真值or相对真值代替
绝对误差&最大允许误差->差值
相对误差&最大引用误差->比值

  1. 绝对误差：测量值与真值之差
$$\mathrm{\Delta}A = A_{X} - A_{0}$$

修正值：$C = - \mathrm{\Delta}A$

理论上，示值+修正值=真值，实际还存在”未定系统误差”未被修正；绝对误差不能反映测量准确度

  1. 相对误差：绝对误差与真值 or 示值之比
$$\gamma_{0} = \frac{\mathrm{\Delta}A}{A_{0}} \times 100\%$$ $$\gamma_{X} = \frac{\mathrm{\Delta}A}{A_{X}} \times 100\%$$

相对误差反映测量的准确度

  1. 最大允许误差：仪器最大绝对误差

模拟仪表：

$$\mathrm{\Delta} = \pm \left( a\% A_{x} + \beta\% A_{m} \right)$$

数字仪表：

$$\mathrm{\Delta} = \pm \left( a\% A_{x} + n个字 \right)$$ $$\mathrm{\Delta} = \pm \left( a\% A_{x} + \beta\% A_{m} + n个字 \right)$$

“字”为分辨力，$a\% A_{x}$为与示值有关的误差，$\beta\% A_{m}$为与量程有关的误差

  1. 最大引用误差：量程范围内最大的绝对误差与量程之比
$$r_{nm} = \frac{|\mathrm{\Delta}A|_{m}}{A_{m}} \times 100\%$$

准确度等级：$G = 最大引用误差的数字部分$

按性质分类的误差

系统误差，随机误差，粗大误差

  1. 系统误差：等精度测无限次平均值与真值之差
$$\varepsilon = {\overline{A}}_{n \rightarrow \infty} - A_{0}$$

表征测量结果偏离真值的程度，反映系统的准确性。

特征：大小、方向恒定不变或按一定规律变化（即可复现or预测）

  1. 随机误差：测量值与等精度测无限次平均值之差
$$\delta_{i} = A_{i} - {\overline{A}}_{n \rightarrow \infty}$$

反映系统的精密性（另一个方面来说反映分散性）

特征：单次测量无规律，多次测量总体服从统计规律

随机误差和系统误差共同反映了系统的精确性（精密性+准确性）

  1. 粗大误差：明显超出统计规律预期值的误差

来源：方法不当、操作疏忽、环境恶化

处理：按一定标准剔除含粗大误差的数据（$3\sigma$准则）

有效数字

  1. 有效数字=确切数字+一位存疑数字
  2. 偶数法则四舍五入：舍去位为5时总使前一位为偶
  3. 误差只取一位有效数字（特殊情况取两位），测量结果末位数与误差末位数对齐

系统误差的处理-校正

两种方法：

  1. 根本上消除系统误差产生的原因
  2. 引入修正值

修正值无法修正的未定系统误差在修正后归为”随机误差”，视完全修正了系统误差

（修正值的获取：更高精度的仪表检定or理论推导or试验）

随机误差的处理-统计学表示

  1. 性质：有界性、居中性、对称性、抵偿性
  2. 分布：正态分布（大样本），t分布（小样本），平均分布（特殊情况）
  3. 数学期望E：无穷次测量平均值

数学期望是真值的最好估计（随差抵偿，还存在系差）

  1. 标准差：测量值与数学期望之差的平方和的平均值的二次根
$$\sigma = \sqrt{\frac{\sum\left( x_{i} - E \right)^{2}}{n}}$$
  1. 相关计算

考虑到”无限次”不具备可操作性，提出
“算术平均值原理”：有限测量列的算数平均值是数学期望的最佳估计。”数学期望-标准差”简化为”算术平均值-实验标准差”

*算术平均值

$$\overline{x} = \frac{\sum x_{i}}{n}$$

*实验标准差

测量列的实验标准差：

$$\sigma(x_{i}) = \sqrt{\frac{\sum\left( x_{i} - \overline{x} \right)^{2}}{n - 1}}$$

算数平均值的实验标准差：

$$\sigma\left( \overline{x} \right) = \sqrt{\frac{\sum\left( x_{i} - \overline{x} \right)^{2}}{n(n - 1)}} = \frac{\sigma\left( x_{i} \right)}{\sqrt{n}}$$
  1. 其他概念

置信概率：真值处于置信区间内的概率

置信区间：$\lbrack E - k\sigma,E + k\sigma\rbrack$，$k$称为置信因子

置信度：表征了多次测量数据的可依赖性，可以理解为置信概率和置信因子的集合

9、测量不确定度及其评定方法

测量报告：$被测量 = 估计值 \pm 测量不确定度U\ \ 单位（置信概率P）$

测量不确定度表征了”测量”这一行为的准确性，测量结果的可信赖性

  1. 测量不确定度的方法

GUM法：对称分布（常用）

MCM法：非对称分布

  1. 两类不确定度

A类不确定度：符合统计规律

$$u_{A} = \sigma(\overline{x})$$

B类不确定度：不符合统计规律

$$u_{B} = \frac{a}{K}$$ $a$为区间半宽度，$K$为置信因子

分布类型的确定：

均匀：不了解被测值分布/误差来源仪器等客观环境/被测值在区间内均匀分布

反正弦：中心概率小，两边概率大

三角：中心概率大，两边概率小

梯形：被测量分布为大小不同的均匀分布之和

正态：给出置信概率和扩展不确定度时/被测量受多个独立随机量影响

  1. 合成不确定度$\mathbf{u}_{\mathbf{C}}$

两种情况，直接/间接测量

*直接测量：

$$u_{C} = \sqrt{u_{A}^{2} + u_{B}^{2}}$$

*间接测量：用输入$x_{i}$的合成不确定度$u_{C}(x_{i})$表示输出$y$的合成不确定度$u_{C}(y)$

两个公式，对应两种测量模型

$Y = f\left( X_{i},X_{i + 1}\ldots \right)$： $$u_{C}(y) = \sqrt{\sum\left( \frac{\partial f}{\partial x_{i}}u_{C}(x_{i}) \right)^{2}}$$ $Y = f(X_{i}^{P_{i}} \cdot X_{i + 1}^{P_{i + 1}}\ldots)$： $$\frac{u_{C}(y)}{\overline{y}} = \sqrt{\sum\left( \frac{P_{i}u_{C}\left( x_{i} \right)}{\overline{x_{i}}} \right)^{2}}$$
