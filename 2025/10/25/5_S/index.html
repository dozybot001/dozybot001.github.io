🚀 Prompt Amplifier (提示词放大器)

把简单的想法，变成 AI 能完美执行的专业指令。

你是否遇到过这种情况：

  • 想让 AI 帮你写代码或写文章，但不知道该怎么描述清楚？
  • 写出的指令太简单，AI 给出的结果总是不尽人意？
  • 一遍又一遍地让AI再次生成，极大地降低了生产效率？

Prompt Amplifier 就是为了解决这个问题而生的。你只需要给出一个模糊的想法，它会像一位专业的咨询顾问，通过几个关键问题引导你完善细节，最后自动生成一份逻辑严密、结构清晰的“生产级”提示词（System Prompt）。

✨ 核心功能

  • 💡 智能引导：你只需说“写个贪吃蛇”或“帮我润色简历”，AI 会自动分析并向你提问（例如：用什么语言？什么风格？面向什么人群？）。
  • 👆 轻松选择：不需要自己绞尽脑汁打字，通过点击卡片选项即可快速明确需求。
  • ➕ 自定义与扩展：觉得选项不够？点击“更多选项”让 AI 挖掘新角度，或者直接输入你的独特要求。
  • 📝 专业合成：一键生成包含角色设定、任务约束、思维链的专业 Prompt，直接复制即可使用。
  • 🔒 隐私安全：你的 API Key 仅存储在你的浏览器/设备本地，不会上传到任何中间服务器。

📖 使用指南

第一步：设置 AI 密钥 (首次使用)

为了让工具运行，你需要配置自己的 AI 模型服务（支持 OpenAI、DeepSeek 等）。

  1. 点击页面右上角的 设置图标 (⚙️)。
  2. API Key: 粘贴你的模型服务商提供的密钥（以 sk- 开头）。
  3. Model (模型): 输入模型名称（例如 gpt-4o 或 deepseek-chat）。
  4. Base URL (选填): 如果使用官方 OpenAI，请留空；如果使用中转服务或 DeepSeek，请输入对应的地址（如 https://api.deepseek.com）。
  5. 点击“保存配置”。

第二步：输入初始想法

在主页底部的输入框中，输入你最原始、最简单的需求。

示例: “帮我写一个 Python 数据分析脚本”

第三步：完善细节

AI 会分析你的指令，并弹出 3 个维度的选项卡（例如：数据源类型、分析目标、图表风格）。

  • 点击选择：点击你想要的标签。
  • 手动输入：如果标签里没有你想要的，可以在输入框填入自定义内容。
  • 更多维度：如果觉得细节还不够，点击 “更多选项”，AI 会继续思考新的补充维度。

第四步：生成与复制

当你对选定的条件满意后，点击 “生成最终 Prompt”。
稍等片刻，屏幕下方会出现生成的完整提示词。点击 “复制 Prompt” 按钮，将其粘贴到 ChatGPT、Claude 或 DeepSeek 中即可使用！

❓ 常见问题 (FAQ)

Q: 为什么点击发送后提示 “API Key Error”?
A: 请检查设置页面的 API Key 是否填写正确，且该 Key 还有余额。

Q: 支持哪些模型？
A: 本工具支持所有兼容 OpenAI 接口格式的模型。推荐使用 gpt-4 系列、claude-3-5 (通过中转) 或 deepseek-chat 以获得最佳的逻辑推理效果。

Q: 我的数据安全吗？
A: 安全。这是一个纯前端运行的应用（或本地客户端），你的 API Key 和聊天记录只保存在你当前的浏览器缓存或设备中，我们无法在后台查看。

Q: 界面支持英文吗？
A: 支持。在设置中可以一键切换 中文 / English 界面。

让 AI 更好地理解你，从这里开始。
