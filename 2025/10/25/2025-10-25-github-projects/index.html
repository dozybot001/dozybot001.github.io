Github Projects

Caret

  1. Description: A lightweight, browser-based AI IDE with Monaco editor, local file system access, and chat-driven code changes. Runs fully in your browser with no installation required.
  2. Repository: https://github.com/dozybot001/Caret/
  3. Website: https://dozybot001.github.io/Caret/

Research on Grid Scheduling Mechanism Design for Virtual Power Plant Cluster

  1. Description: Undergraduate graduation thesis at Shanghai Jiao Tong University focusing on grid scheduling mechanism design for virtual power plant clusters, with emphasis on subject differentiation, regulation quality evaluation, market clearing mechanisms, and cluster scheduling frameworks.
  2. Repository: https://github.com/dozybot001/Graduation_Thesis/

EE3321 Course Design in Intelligent Instrument

  1. Description: Comprehensive coursework project demonstrating skills in intelligent instrument design, covering PCB design (Altium Designer), virtual instrument development (LabVIEW), and embedded systems programming (MSP430). Awarded grade 98/100, ranked #1 among peers.
  2. Repository: https://github.com/dozybot001/Instrument_Design/

Automatic Page Turning Device

  1. Description: Coursework submission for SJTU SI1210 Engineering Practice. An automatic page turning device with CAD design, voice recognition module, and mechanical assembly. Awarded Grand Prize of Student Innovation Competition (Team Project).
  2. Repository: https://github.com/dozybot001/page_turning_machine/

Sensor-Based Automatic Water-Stopping Water Dispenser

  1. Description: Coursework submission for SJTU ME1221 Introduction to Engineering. Features automatic water level detection, cup measurement, voice recognition, and gradual water stopping mechanism. Awarded TOP 10 Projects (Highest award at university level).
  2. Repository: https://github.com/dozybot001/smart_water_dispenser/

Data Structures Laboratory

  1. Description: A comprehensive C++ implementation of fundamental data structures and algorithms with an interactive menu-driven interface. Includes lists, stacks, queues, trees, hash tables, graphs, priority queues, sets, and various algorithms.
  2. Repository: https://github.com/dozybot001/data_structure/

Embedded Systems Projects

  1. Description: Embedded systems projects based on coursework assignments, including stepper motor control, GPIO operations, SysTick timer, UART/I2C communication, PWM, and major assignment implementations for TM4C microcontrollers.
  2. Repository: https://github.com/dozybot001/embedded_systems/

Substation Design Project

  1. Description: Electrical substation design coursework including transformer selection, short-circuit current calculation, maximum current analysis, and comprehensive system design with Python-based calculation scripts.
  2. Repository: https://github.com/dozybot001/substation_design/

Auto Space

  1. Description: Automatically adds spaces between Chinese characters and Latin characters/punctuation in text files. Based on daft-auto-spacing project.
  2. Repository: https://github.com/dozybot001/auto_space/
  3. Website: https://dozybot001.github.io/auto_space/

Pure Color

  1. Description: A color picker tool with fullscreen display capability, useful for viewing pure color backgrounds (e.g., for scanning X-ray films).
  2. Repository: https://github.com/dozybot001/pure_color/
  3. Website: https://dozybot001.github.io/pure_color/

Prompt Amplifier

  1. Description: A Flutter-based prompt amplifier tool that transforms simple ideas into professional AI instructions through interactive guidance.
  2. Repository: https://github.com/dozybot001/prompt_amplifier_flutter/
